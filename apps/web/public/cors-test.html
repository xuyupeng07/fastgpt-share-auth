说明

  • CORS 是浏览器安全机制：服务器可以响应任何请求，但浏览器会根据 CORS 头部决定是否允许 JavaScript 访问响应。
  • curl 等工具不受 CORS 限制：CORS 只在浏览器环境中生效。
  • 预检请求：对于复杂请求，浏览器会先发送 OPTIONS 请求检查权限。
  • 当前配置：只允许 https://demo.fastgpt.cn 域名访问 API。
